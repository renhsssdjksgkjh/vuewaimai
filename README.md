# vue-waimai
基于 Vue3 打造的仿美团外卖移动端项目，还原外卖平台核心交互与视觉效果，覆盖首页、订单页、我的、购物车、地址管理等核心业务场景，适合 Vue3 移动端实战学习。

## 项目介绍
`vue-waimai` 是一套面向前端学习者的仿美团外卖实战项目，基于 Vue3 生态开发，重点实现外卖平台的核心功能逻辑，同时兼顾代码规范和移动端适配，可作为 Vue3 移动端实战入门的参考项目。

## 技术栈
| 类别         | 技术选型                                  |
|--------------|-------------------------------------------|
| 核心框架     | Vue3 (Setup 语法糖 + `<script setup>`)    |
| 构建工具     | Vue CLI（基于 Webpack）                   |
| 状态管理     | Vuex 4 + localStorage（状态持久化）       |
| UI 组件库    | Vant 4（适配移动端的轻量组件库）          |
| 网络请求     | Axios（接口封装、拦截器）                  |
| 路由管理     | Vue Router 4（路由懒加载、路由守卫）      |
| 样式处理     | Less（样式预处理、变量复用）              |
| 移动端适配   | amfe-flexible + postcss-pxtorem           |
| 代码规范     | ESLint + Prettier（统一代码风格）         |

## 核心功能
✅ **首页模块**：定位切换、商家分类筛选、推荐商家列表、关键词搜索、活动专区（天天神券/减配送费等）
✅ **购物车模块**：商品管理、数量修改、价格自动计算、结算操作
✅ **订单模块**：模拟下单、订单生成、订单列表与详情展示
✅ **地址管理**：收货地址增删改查、默认地址设置
✅ **个人中心**：登录/注册、个人信息编辑、设置入口
✅ **状态管理**：使用 Vuex + localStorage 实现全局状态管理与数据持久化

## 环境要求
- Node.js ≥ 14.0.0（推荐 16+ 稳定版）
- 包管理器：Yarn（项目指定）
- 浏览器：Chrome/Firefox/Edge（最新版，适配移动端调试）
